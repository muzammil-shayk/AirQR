# 🚀 AirQR - Modern QR Code Generator

A secure, fast, and privacy-focused QR code generator built with Next.js 15, TypeScript, and Tailwind CSS. Generate QR codes instantly with advanced customization options.

[![Public API](https://img.shields.io/badge/API-Public-teal)](https://airqr.vercel.app/api-docs)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## 🌐 Developer API (Free & Public)

AirQR provides a lightning-fast, public API to generate extremely crisp SVG QR codes instantly without any authentication! Perfect for use in your own apps, websites, or CLI tools.

- **Base Endpoint**: `https://airqr.vercel.app/api/qr`
- **Features**: **No Auth**, **HTTPS-only**, **Full CORS support**, **Native SVG**.
- **Example Usage**:
  ```html
  <img src="https://airqr.vercel.app/api/qr?data=Hello%20World" width="200" />
  ```

[**View Full API Documentation →**](https://airqr.vercel.app/api-docs)

## ✨ Features

### Core Functionality

- **🎯 Multiple QR Types**: URL, Text, Email, Phone, SMS, WiFi, vCard
- **🎨 Full Customization**: Colors, sizes, error correction levels, margins
- **📱 Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **⚡ Real-time Preview**: Instant QR code generation as you type
- **⌨️ Keyboard Shortcuts**: Press **Enter** to generate instantly from any input

### Advanced Features

- **📦 Multiple Formats**: PNG, SVG download options
- **📋 Easy Sharing**: Copy text, share QR codes natively
- **🏷️ Branding**: Minimalist "Generated by AirQR" watermark on all outputs

### Security & Privacy

- **🔒 Client-side Processing**: No data sent to servers (except for the public API optionally)
- **🛡️ Content Security Policy**: Strict CSP headers for security
- **🚫 Input Sanitization**: XSS protection and validation
- **⚖️ Rate Limiting**: Protection against abuse
- **🔐 Secure Headers**: HSTS, X-Frame-Options, and more

## 🛠️ Tech Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **QR Generation**: QRCode.js
- **Icons**: Lucide React, React Icons
- **Deployment**: Vercel

## 🚀 Quick Start

### Prerequisites

- Node.js 20.x or later
- npm, yarn, or pnpm

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/muzammil-shayk/airqr.git
   cd airqr
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**
   Visit [http://localhost:3000](http://localhost:3000)
